# ALMA — Visão Geral

## O que é o ALMA

**ALMA não é um app de meditação.**

ALMA é o primeiro app brasileiro que usa **inteligência artificial** para criar uma jornada de autoconhecimento completamente personalizada para cada utilizador.

O nome não é acidente. *Alma* em português significa soul — e é exactamente isso que o app busca: ajudar cada pessoa a encontrar a sua alma, o seu destino, o seu propósito de vida.

---

## O Diferencial Real

Enquanto outros apps oferecem meditações genéricas, o ALMA aprende quem você é:

| Input | O que aprende |
|-------|--------------|
| Data de nascimento | Padrões numerológicos, ciclos de vida |
| Conexão Facebook/Instagram | Interesses, gostos, rede social |
| Comportamento no app | Padrões, humor, consistência |
| Histórico de uso | Evolução ao longo do tempo |

**Output:** Uma experiência única — como se o app te conhecesse há anos.
